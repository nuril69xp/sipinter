Petunjuk Penggunaan
Cara menggunakan Si-PINTER
📋 Cara Meminjam Perangkat
1
Buka menu Pinjam di navigasi bawah
2
Tap tombol Scan QR Code lalu arahkan kamera ke stiker QR pada perangkat
3
Atau ketik kode aset secara manual (contoh: CB-01, PJ-02)
4
Pilih nama guru dan kelas/ruang dari dropdown
5
Klik Ambil Perangkat — data otomatis tersimpan ke Google Sheets
🔍 Cara Cek Status Perangkat
1
Buka menu Aset untuk melihat semua perangkat
2
Gunakan filter Tersedia / Dipinjam / Rusak untuk menyaring
3
Tap nama perangkat untuk melihat detail lengkap
📱 Install di Layar Utama HP
1
Buka link Si-PINTER di browser Chrome
2
Tap menu titik tiga (⋮) di pojok kanan atas Chrome
3
Pilih Tambahkan ke layar utama
4
Ikon Si-PINTER muncul di HP seperti aplikasi sungguhan!
